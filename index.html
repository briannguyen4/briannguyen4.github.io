Stack Overload


Stack Overload is a single-page, full-stack application where users can create questions as well as answer and vote on questions. Based on the website Stack Overflow, it was built using Ruby on Rails and a PostgreSQL database on the backend while implementing React and Redux on the frontend.

Live Site   |   Github

TypeTempo


TypeTempo is a typing game developed with vanilla Javascript, HTML5, and CSS3. This game tests how fast a player can accurately type through various text snippets. At the end of the game, results are generated including information such as WPM.

Live Site   |   Github

Deception


Online adaptation of the board game Deception: Murder in Hong Kong. This social game involves players having specific roles and having to deduce who the murderer is based off of evidence cards. Utilization of websockets allows for players on different browsers to play in real-time.
